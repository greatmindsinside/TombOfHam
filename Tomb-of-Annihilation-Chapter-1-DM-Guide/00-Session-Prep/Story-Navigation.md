# Story Navigation: Choose Your Own Adventure Guide

*Official source: Chapter 1: **pp. 15-36***  
*Reference: [D&D 5E - Tomb of Annihilation.pdf](../../D&D%205E%20-%20Tomb%20of%20Annihilation.pdf)*

*This document provides a comprehensive navigation system for Chapter 1, allowing DMs to quickly find the next scene based on player choices and story flow.*

## How to Use This Guide

This navigation system works like a "choose your own adventure" book. Follow the decision points based on what your players choose to do. Each path leads to the relevant document and section.

---

## Story Flow Overview

```text
START: Story Setup
  ↓
[Players Accept Quest?]
  ├─ YES → Syndra Meeting → Teleportation → Arrival in Port Nyanzaru
  └─ NO → Handling Player Reluctance → (eventually) Story Setup
  ↓
Arrival in Port Nyanzaru
  ↓
[Choose Path: Fast-Track OR Full Experience]
  ├─ Fast-Track → One Social Scene → Guide Selection → Gear Purchase → Departure
  └─ Full Experience → City Exploration → Multiple Activities → Preparation → Departure
  ↓
Departure → Chapter 2: The Land of Chult
```

## Common Navigation Patterns

### Pattern 1: Information Gathering Loop

**If players want to gather information:**

1. Start at [City-Overview.md](../02-Arrival-in-Port-Nyanzaru/City-Overview.md)
2. Choose information source:
   - **Tavern rumors** → [Tavern-Rumors.md](../04-Side-Quests-and-Activities/Tavern-Rumors.md)
   - **Meet Volo** → [Meeting-Volo.md](../03-City-Exploration-and-Social-Play/Meeting-Volo.md)
   - **General gathering** → [Gathering-Information.md](../03-City-Exploration-and-Social-Play/Gathering-Information.md)
   - **NPCs** → [Key-NPCs.md](../03-City-Exploration-and-Social-Play/Key-NPCs.md)
3. After gathering information, check if they have a solid lead:
   - **Yes** → Continue to preparation
   - **No** → Return to information gathering or → [Solid-Lead-Requirements.md](../06-Chapter-Transition/Solid-Lead-Requirements.md)

### Pattern 2: Preparation Loop

**If players are preparing for the jungle:**

1. Start at [City-Overview.md](../02-Arrival-in-Port-Nyanzaru/City-Overview.md)
2. Choose preparation task:
   - **Hire guide** → [Hiring-Guides.md](../05-Outfitting-and-Preparation/Hiring-Guides.md)
   - **Buy supplies** → [Supplies-and-Gear.md](../05-Outfitting-and-Preparation/Supplies-and-Gear.md)
   - **Get charter** → [Expedition-Charter.md](../05-Outfitting-and-Preparation/Expedition-Charter.md)
3. After each task, check readiness:
   - **Ready** → [Chapter-Exit-Conditions.md](../06-Chapter-Transition/Chapter-Exit-Conditions.md)
   - **Not ready** → Continue preparation

### Pattern 3: Side Quest Loop

**If players want to take side quests:**

1. Start at [City-Overview.md](../02-Arrival-in-Port-Nyanzaru/City-Overview.md)
2. Go to [Side-Quests.md](../04-Side-Quests-and-Activities/Side-Quests.md)
3. After completing quest(s), return to:
   - **Information gathering** → [Gathering-Information.md](../03-City-Exploration-and-Social-Play/Gathering-Information.md)
   - **Preparation** → [Hiring-Guides.md](../05-Outfitting-and-Preparation/Hiring-Guides.md) or [Supplies-and-Gear.md](../05-Outfitting-and-Preparation/Supplies-and-Gear.md)
   - **More quests** → [Side-Quests.md](../04-Side-Quests-and-Activities/Side-Quests.md)

## Quick Reference: Scene Transitions

### From Story Setup

- **Players accept** → Syndra Meeting
- **Players decline** → Handling Player Reluctance

### From Syndra Meeting

- **Players accept** → Teleportation to Chult
- **Players negotiate** → Teleportation to Chult (with benefits)

### From Teleportation

- **Always** → Harbor Description → City Overview

### From City Overview

- **Fast-Track** → Fast-Track Chapter 1
- **Full Experience** → Multiple options (see Decision Point 4)

### From Any City Activity

- **Ready to prepare** → Hiring Guides or Supplies and Gear
- **Want more exploration** → Return to City Overview
- **Ready to leave** → Chapter Exit Conditions

### From Preparation

- **Have everything** → Chapter Exit Conditions
- **Missing items** → Return to relevant preparation document

### From Chapter Exit Conditions

- **Ready** → Leaving Port Nyanzaru
- **Not ready** → Return to missing requirement document

## Navigation Tips

1. **Bookmark this page** - Use it as your central navigation hub
2. **Follow player choices** - Let players drive the story, use this guide to find the next scene
3. **Don't force paths** - Players can jump between activities; use this guide to find where they want to go
4. **Use Fast-Track when needed** - If time is limited, switch to Fast-Track path
5. **Check readiness often** - Use Chapter Exit Conditions to verify players are ready to leave
