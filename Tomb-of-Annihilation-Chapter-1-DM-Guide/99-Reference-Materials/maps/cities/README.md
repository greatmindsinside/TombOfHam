# City Maps

This folder contains maps for cities, towns, and settlements.

## Usage

City maps are useful for:

- Urban exploration
- Social encounters
- Navigation and travel planning
- Combat in city streets
- Finding specific locations

## Map Types

- Full city overviews
- District maps
- Street-level maps
- Building exteriors
- Marketplaces and plazas

## File Organization

Organize by:

- City name
- District or area
- Scale (overview vs detailed)

## Tips

- City maps can usually be shown to players (they're exploring, not sneaking)
- Use district maps for detailed exploration
- Overview maps help with navigation and planning
- Consider both player and DM versions if maps contain hidden information
