## Usage Tips

- **Grid versions**: Use for precise combat encounters and movement tracking
- **Gridless versions**: Use for more flexible gameplay or when using custom grid systems
- **Multi-level taverns**: Great for complex encounters with vertical movement
- **Single-level taverns**: Perfect for quick social encounters

## File Naming Convention

Files follow this pattern:

- `[tavernname]_[creator]/[tavernname]_[creator]_level-[X]_grid.jpg` - Grid version
- `[tavernname]_[creator]/[tavernname]_[creator]_level-[X].jpg` - Gridless version

## When to Use

- Social encounters in Port Nyanzaru
- Information gathering scenes
- Resting locations
- Meeting NPCs
- Combat encounters in urban settings
- Player character backstories involving taverns
