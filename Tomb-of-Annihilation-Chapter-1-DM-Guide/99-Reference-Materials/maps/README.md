# Maps Directory

This directory contains all map files organized by type for easy access during your Tomb of Annihilation campaign.

## Folder Structure

- **taverns/** - Tavern and inn maps for social encounters
- **official/** - Official Tomb of Annihilation maps (if you have local copies)
- **dungeons/** - Dungeon and underground complex maps
- **cities/** - City and settlement maps
- **locations/** - Specific location maps (forts, camps, ruins, etc.)
- **player-handouts/** - Maps safe to show players (no hidden information)

## Usage

Each folder contains maps relevant to that category. See the README in each subfolder for details about available maps.

## Map File Formats

Maps are typically provided in:

- **JPG/PNG** - Standard image formats for most VTT platforms
- **Grid versions** - Maps with grid overlays for precise movement
- **Gridless versions** - Maps without grids for flexible use

## Adding New Maps

When adding new maps:

1. Place them in the appropriate category folder
2. Use descriptive filenames
3. Update the folder's README.md with the new map information
4. Consider adding both grid and gridless versions if available
