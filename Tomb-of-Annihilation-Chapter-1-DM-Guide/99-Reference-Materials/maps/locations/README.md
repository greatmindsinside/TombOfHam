# Location Maps

This folder contains maps for specific locations such as forts, camps, ruins, and other notable sites.

## Usage

Location maps cover specific areas that aren't full cities or dungeons but are important campaign locations.

## Map Types

- Forts and strongholds
- Camps and outposts
- Ruins and ancient sites
- Temples and shrines
- Natural landmarks
- Special structures

## File Organization

Organize by:
- Location name
- Type (fort, camp, ruin, etc.)
- Grid vs gridless versions

## Tips

- Some location maps may contain hidden information (traps, secret areas)
- Others can be shown to players directly
- Consider the location's purpose when deciding player visibility
- Use appropriate scale for the location type
