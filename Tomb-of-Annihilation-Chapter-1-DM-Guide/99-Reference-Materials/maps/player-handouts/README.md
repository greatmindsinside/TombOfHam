# Player Handout Maps

This folder contains maps that are safe to show players. These maps do not contain hidden information such as traps, secret doors, or monster locations.

## Usage

Player handout maps are designed to be shared directly with players. They help with:

- Navigation and travel planning
- Understanding location layouts
- Visualizing the world
- Planning routes and strategies

## Map Types

- Regional overview maps
- City layouts (without hidden information)
- Public building layouts
- Travel route maps
- Exploration maps

## Important Notes

- **No Hidden Information**: These maps should not reveal traps, secret doors, or monster locations
- **Player-Safe**: All maps in this folder can be shown to players without spoilers
- **DM Versions**: Keep DM versions with hidden information in other folders

## File Organization

Organize by:

- Map type (regional, city, location)
- Purpose (navigation, exploration, etc.)

## Tips

- Use these maps for player reference during sessions
- Print or share digitally as needed
- Update as players discover new areas
- Consider using these for player notes and planning
