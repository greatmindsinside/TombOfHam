# Official Maps

This folder is for storing local copies of official Tomb of Annihilation maps from the sourcebook.

## Purpose

If you have digital copies of official maps (scanned or downloaded), place them here for easy access during gameplay.

## Official Map Types

- Regional maps (Chult overview)
- City maps (Port Nyanzaru, Omu)
- Dungeon maps (Fane of the Night Serpent, Tomb of the Nine Gods)
- Location maps (Fort Beluarian, Camp Vengeance, etc.)

## File Organization

Organize official maps by:
- Chapter or location
- Map type (regional, city, dungeon, location)
- Player vs DM versions (if applicable)

For page references and online links to official maps, see the [Maps Reference Guide](../../Maps.md).
